# Persona-Adaptive Customer Support Agent

## Overview
This project is a simple AI-based customer support chatbot that:
- Detects user persona (technical, frustrated, business)
- Retrieves answers from a knowledge base (KB)
- Adapts response tone based on persona
- Escalates to a human agent if no answer is found

---

## Tech Used
- Node.js
- Express.js
